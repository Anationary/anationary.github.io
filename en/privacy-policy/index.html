3. Cookies information.

  1. Website take advantage of cookie files.
  2. Cookie files consist of IT data, specifically text files, which are contained in the end device of the user and are intend to use by the web pages. Cookies usually contain name of the website which comes from, time of keeping on the end device and unique id number.
  3. Entity creating and retrieving cookies on the user’s end device is Operator of website.
  4. Cookie files are created in order to:
    1. gathering statistics, which helps to understand, how the users are making use of the site;
    2. upkeep session of the user (after logging in);
    3. defining user profile in order to display properly matched contextual ads in ad services, specifically Google AdSense.
  5. On the website are used two basic types of cookies: session and persistent cookies. Session cookies are temporary files that are stored in the end device of a user to log out time, leave the website or disable the software (web browser). Persistent cookies are stored inthe end device of a user for the time specified in the parameters of cookies or until their removal by the user.
  6. Software to browse the web (browser) usually by default allows storage of cookies in the end device of a user. Website users can change the settings in this area. Browser allow to delete cookies. You can also automatically block cookies, see the help or documentation of your web browser.
  7. Restrictions on the use of cookies may affect some of the functionality available on the website.
  8. Cookies posted at the end device of the user may be used also by cooperating with the Operator’s advertisers and partners.
  9. We recommend that you read the privacy policies of these companies to learn how to use the cookies used in statistics: „Privacy policy of Google Analytics”.
  10. Cookies can be used by ad networks, in particular, the Google to display ads tailored to the way you use the website. For this purpose, can maintain information about the user navigation path or the time remaining on the page.
  11. In terms of information about user preferences collected by the Google Ads, you can view and edit information derived from cookies using the tool: „https://www.google.com/ads/preferences/”.
